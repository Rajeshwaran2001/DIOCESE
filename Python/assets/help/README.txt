Help screenshots
================
Drop PNG screenshots here to make them appear on the in-app Help pages
(Menu bar -> Help). If a file is missing, a grey placeholder is shown instead,
so the app works fine without them.

Expected filenames (one per Help topic):
  overview.png   - the main window / sidebar
  death.png      - the Death Extract screen
  marriage.png   - the Marriage Returns screen
  baptism.png    - the Baptism Certificate screen
  settings.png   - the Settings screen

Tip: take the screenshots at ~1100px wide; they are scaled down to fit.
